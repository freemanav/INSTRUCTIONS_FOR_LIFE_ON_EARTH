3. WHAT FORMATS ARE INCLUDED AND WHY THEY
──────────────────────────────────────────────────────

For each language you will find 3 (+1 in Russian) main formats + font + instructions.

*** Formats and Why:
──────────────────
1. DOCX format
- Original Word file.
- Inside - "no page break" mode.
- The book is presented as a continuous text, like a web page.
- From this file you can create any other format, print the book, edit the translation.
- This is the most flexible format.
──────────────────
2. HTML format
- Web page. Opens in any browser.
- Also a continuous, "river" text, without page breaks.
- Ideal for reading from the screen of a phone, tablet, or computer.
- You can make a PDF from it (print → save as PDF).
──────────────────
3. Format TXT
- Plain text without formatting. UTF-8 encoding.
- This is the MOST ETERNAL FORMAT.
- Will open in any text editor on any OS.
- Even if everything else doesn't open in 100 years, TXT will always open. It's your insurance.
──────────────────
4. PDF format (available only in the Russian version of the book).
- Also one of the long-lasting formats.
- Convenient for printing.
- IMPORTANT: For the PDF to display correctly, the same font (Verdana) is required. If the font is missing, the text may be distorted. Therefore, the font is included separately.
- In PDF, you can also set the "continuous" mode (no page breaks).
──────────────────

5. VERDANA FONT (WHY IS IT APPLIED)
──────────────────────────────────────────────────────

The book was created using the Verdana font, which is the standard Windows font found on most computers.

BUT: in 50-100 years, this font may no longer be available in new operating systems. Or it may be replaced by another.

Therefore, the ORIGINAL Verdana.ttf font file was included in the “Font_Verdana” folder along with installation instructions.

If you see gibberish, strange symbols, or text that doesn't look as intended when opening HTML, DOCX, or TXT (and PDF), install this font.

HOW TO INSTALL A FONT:

WINDOWS:
1. Right-click on the Verdana.ttf file
2. Select "Install"
3. Or copy it to the C:\Windows\Fonts\ folder

macOS:
1. Double-click the Verdana.ttf file
2. Click "Install Font"
3. Or copy to /Library/Fonts/ or ~/Library/Fonts/

LINUX:
1. Copy the file to /usr/share/fonts/ or ~/.fonts/
2. Run the command: fc-cache -fv
3. Restart the program in which you open the book

After installation, restart your browser or Word. Everything should display correctly.
correctly.

8. HOW TO PRINT A BOOK
──────────────────────────────────────────────────────

All files in this archive are in "no page breaks" mode (solid text).
This is done intentionally: the book is conceived as a “river” text.

If you want to print the book:

Option 1 (easier):
- Open the HTML file in your browser.
- Press Ctrl+P (or Cmd+P on Mac).
- In the print settings, select the desired page size, margins, and headers and footers.
- Print it.

Option 2 (more flexible):
- Open the DOCX file in Word (or LibreOffice, Google Docs).
- Add page breaks yourself where you think necessary.
- Print it out.

The book isn't bound by rigid layout. You are the master of how you read and print it.
